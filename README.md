# 🧩 RubiksCubeSolver

A Java-based **3×3 Rubik's Cube Solver** built from scratch.

This project is a personal learning journey to understand object-oriented programming, algorithms, and Rubik's Cube mechanics by implementing a complete cube simulator and solver in Java.

---

## 🚀 Features

### Current

* Read all six faces of a 3×3 Rubik's Cube
* Store the cube using a dedicated `Cube` class
* Print cube faces
* Modular project structure using separate Java classes

### Planned

* Input validation
* Face rotations (U, D, L, R, F, B)
* Counter-clockwise and double moves
* Cube state validation
* Complete solving algorithm
* Move optimization
* Scramble generator
* Interactive command-line interface
* Web interface

## ▶️ Running the Project

1. Clone the repository.
2. Open it in VS Code or your preferred Java IDE.
3. Compile and run `Main.java`.
4. Enter the six faces of the cube using space-separated color letters.

Example input:

```text
W W W
W W W
W W W
```
